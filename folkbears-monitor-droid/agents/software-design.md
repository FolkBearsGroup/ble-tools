# 設計

## 目的

各種の BLE アドバタイズを受信してモニタリングするツール

- iBeacon の発見
- FolkBears の発見
- EN api モードの発見
- その他のBLE アドバタイズ

受信のみで、動作チェックに使う。
デバイス名発見など
大体の距離で機器の場所を確認は要検討

## 仕様

1. タブで表示切り替え
2. 5分以前のデータは消える
3. 距離順でソートが可能
4. 受信回数でソートが可能
5. 特定受信デバイスをマークできる。 Mac アドレスでチェックする。
6. 画面を10秒間隔で更新
7. 画面更新の停止モードで、チェックマークなどで確認が可能

これを、iBeacon版、FolkBears 版、EN API 版を作る


## 設定

- 設定の手入力（スマホの画面で設定）
- 設定を json 形式で貼り付けが可能

## 実装技術

- Android アプリ(Kotlin)
- Jetpack Compose で UI を実装
- BLE アドバタイズの受信
 - BeaconScan クラスの利用
 - ENSimScan クラスの利用
 - GattClient ではデバイスの発見までを利用する
- タブの実装
